# Analiza sprzedaży suplementów - dashboard

## Opis projektu
Projekt polegał na stworzeniu interaktywnego dashboardu z użyciem ```Looker studio```. Projekt analizuje sprzedaż suplementów w e-commerce (Amazon, Walmart, iHerb) w krajach Ameryki Północnej. Dane obejmują:
- przychody (Revenue),
- liczbę sprzedanych jednostek (Units Sold),
- rabaty (Discount),
- zwroty (Units Returned),
- kategorię produktu,
- platformę sprzedaży,
- czas (Date).
